# **App Name**: Zintl

## Core Features:

- Smart Document Intelligence: Utilizes AI as a tool to summarize complex PDF contents, extract key data points, and answer contextual questions directly from the file.
- Rapid Multi-Format Processing: Instant batch processing for PDF and Image compression, conversion (PDF to JPG, JPG to PDF, Word to PDF), and reorganization using an edge-optimized worker system.
- AI Vision Document Scanner: Uses computer vision and OCR to enhance scanned document images, perform intelligent cropping, and generate text-searchable PDFs from photos.
- Visual Workflow & Glassmorphic Preview: A responsive drag-and-drop workspace with real-time before/after processing previews and visual rearranging of document pages.
- Dynamic Cloud Workspace: Seamlessly manages user document preferences and temporary workspace state using Firestore for persistent but ephemeral document metadata.
- SEO-Architected Page Engine: Programmatic generation of highly-optimized landing pages for each specific tool type (e.g., /compress-pdf) featuring dynamic JSON-LD schemas and FAQ snippets.
- Context-Aware Ad Integration: Unobtrusive, intelligently-placed Google AdSense placeholders optimized for Gen Z engagement without interrupting the document workflow.

## Style Guidelines:

- The palette is based on a 'Digital Neo-Minimalism' anchor. Primary color is a vibrant electric indigo (#818CF8) to evoke intelligence. The background is a sophisticated deep charcoal (#0F172A), providing a premium high-contrast feel. The accent is a sleek cyan-blue (#2DD4BF) used for focus points and success states.
- A sophisticated font pairing of 'Space Grotesk' for bold, futuristic headlines and 'Inter' for high-readability body text to satisfy professional use cases. Code snippets use 'Source Code Pro'.
- Bento-style thick outline icons with subtle gradients to match the modern SaaS look, emphasizing tool functions over literal document representations.
- A card-based glassmorphism layout with heavy use of blur backgrounds and generous white space (padding), specifically designed for a mobile-first touch experience.
- Utilizes subtle micro-interactions like soft bounces on drag-and-drop actions and fluid SVG progress bars during file processing phases.
